## 六、功能实现详解

### 6.1 地图初始化与多底图切换（第08讲练习2）

**功能描述**：初始化 Leaflet 地图实例，加载天地图/OSM 多种底图，通过图层控件实现切换。

**技术实现**：
- 使用 `L.map()` 创建地图实例，初始中心为北京天安门 `[39.9042, 116.4074]`，缩放级别 11
- 天地图瓦片通过 WMTS 接口加载：`http://t{s}.tianditu.gov.cn/{layer}_w/wmts`
- 支持 4 种底图：矢量地图（vec+cva 注记）、卫星影像（img+cia 注记）、地形地图（ter+cta 注记）、OpenStreetMap
- 使用 `L.layerGroup` 将底图与注记绑定为一个逻辑图层组
- 通过 `L.control.layers(baseLayers, overlays)` 创建图层控制器

**关键代码**：
```javascript
// 底图组合（底图 + 对应注记包装为 LayerGroup）
const baseVecGroup = L.layerGroup([tiandituVec, tiandituCva]);
const baseImgGroup = L.layerGroup([tiandituImg, tiandituCia]);
const baseTerGroup = L.layerGroup([tiandituTer, tiandituCta]);
const baseOSMGroup = L.layerGroup([osmLayer]);

// 图层控制器
L.control.layers(baseLayers, overlayLayers, {
    position: 'topright', collapsed: false
}).addTo(map);
```

### 6.3 地图书签/视角定位（第08讲挑战1）

**功能描述**：用户可保存当前地图视角为书签，支持一键跳转、删除管理，书签数据持久化到 localStorage。

**技术实现**：
- 使用 `localStorage` 存储书签数据（名称、经纬度、缩放级别、创建时间）
- `map.getCenter()` 和 `map.getZoom()` 获取当前视角
- `map.flyTo()` 实现平滑飞行动画跳转（duration: 1.5s）
- 左侧面板实时显示当前视角坐标和缩放级别
- 回车键快捷保存，localStorage 持久化存储

**关键代码**：
```javascript
function addBookmark() {
    const center = map.getCenter();
    const zoom = map.getZoom();
    bookmarks.push({
        name: name,
        lat: center.lat, lng: center.lng,
        zoom: zoom,
        createdAt: new Date().toISOString()
    });
    saveBookmarks(bookmarks);
}

function flyToBookmark(index) {
    const bm = bookmarks[index];
    map.flyTo([bm.lat, bm.lng], bm.zoom, {
        animate: true, duration: 1.5
    });
}
```

### 6.4 比例尺、坐标显示与距离测量（第09讲练习2）

**功能描述**：
- 比例尺控件（公制，左下角）
- 实时坐标显示（鼠标移动 + 点击坐标，底部状态栏）
- 距离测量工具（Shift+点击添加测量点，支持拖拽调整、撤销、清除）

**技术实现**：

**（a）比例尺**：
```javascript
L.control.scale({ metric: true, imperial: false, position: 'bottomleft' }).addTo(map);
```

**（b）坐标显示**：
- 底部坐标栏实时显示：经度、纬度（mousemove 事件）、缩放级别（zoomend 事件）、鼠标点击坐标（click 事件）
- 使用 `e.latlng.lat.toFixed(6)` 保留 6 位小数

**（c）距离测量**：
- 使用 Haversine（半正矢）公式计算球面距离，地球半径取 6371000 米
- Shift+点击地图添加测量点（避免与普通地图交互冲突）
- 测量点可拖拽调整位置（`draggable: true` + `drag` 事件更新）
- Ctrl+Z 快捷键撤销上一个测量点
- 线段距离标签显示在每段中点位置
- 起点绿色标记、终点红色标记，中间点蓝色

**Haversine 公式**：
```javascript
function haversineDistance(latlng1, latlng2) {
    const R = 6371000; // 地球平均半径（米）
    const dLat = (latlng2.lat - latlng1.lat) * Math.PI / 180;
    const dLng = (latlng2.lng - latlng1.lng) * Math.PI / 180;
    const a = Math.sin(dLat/2)**2 +
              Math.cos(lat1)*Math.cos(lat2)*Math.sin(dLng/2)**2;
    const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
    return R * c;
}
```

#### 分级设色图（Choropleth Map）

**分级策略**：自定义 8 级断点（0, 10000, 20000, 35000, 55000, 80000, 100000, 135000），指数型递增间隔，低值区密集、高值区稀疏。

**配色方案**：**ColorBrewer YlOrRd 顺序色带**（黄→橙→红 8 级）

| 色阶 | 颜色值 | GDP范围（亿元） |
|:---:|--------|:-------------:|
| L1 | `#FFEDA0` 浅黄 | < 10,000 |
| L2 | `#FED976` 亮黄 | 10,000~20,000 |
| L3 | `#FEB24C` 浅橙 | 20,000~35,000 |
| L4 | `#FD8D3C` 橙色 | 35,000~55,000 |
| L5 | `#FC4E2A` 橙红 | 55,000~80,000 |
| L6 | `#E31A1C` 红色 | 80,000~100,000 |
| L7 | `#BD0026` 深红 | 100,000~135,000 |
| L8 | `#800026` 暗红 | > 135,000 |

**配色选择依据**：
1. GDP 是**连续数值型数据**，有明确的低→高方向性 → 选用顺序色带（Sequential）
2. 顺序色带通过**亮度递减 + 饱和度递增**编码数值：浅色=低值、深色=高值，符合"深色=更多"的视觉直觉
3. **不选用发散色带**（如 RdBu）：GDP 没有自然中心值或正负对称关系
4. **不选用定性色带**（如 Set1）：定性色带适合分类数据，用于连续数值会误导读者

**核心 API 使用**：

```javascript
choroplethLayer = L.geoJSON(geoJSONData, {
    style: choroplethStyle,        // style 回调 → 按GDP返回 fillColor
    onEachFeature: onEachProvince, // onEachFeature → 绑定 hover/click/popup
    filter: filterHighGDP          // filter 选项 → 过滤高GDP省份（可选）
}).addTo(map);
```

#### 比例符号图（Proportional Symbol Map）

**缩放公式**：$r = \sqrt{population} \times 1.5$（平方根缩放，Flannery 感知校正）

**选择平方根缩放的原因**：
- 人眼比较圆形面积时存在系统性低估（感知偏差约 0.7 次幂）
- 线性缩放会导致大值符号面积过于庞大（2000万:200万 = 100:1）
- 平方根将比例压缩到约 10:1，视觉上可区分且不至于过于悬殊

**核心 API 使用**：

```javascript
symbolLayer = L.geoJSON(citiesGeoJSON, {
    pointToLayer: function(feature, latlng) {
        const radius = Math.sqrt(feature.properties.population) * 1.5;
        return L.circleMarker(latlng, { radius: radius, ... });
    },
    onEachFeature: onEachCity
}).addTo(map);
```

| 模式 | 技术实现 | 效果 |

| 仅分级设色 | `map.removeLayer(symbolLayer)` | 仅省级GDP色块 |
| 仅比例符号 | `map.removeLayer(choroplethLayer)` | 仅城市人口圆 |
| 高GDP省份 | `filter: f => gdp >= 50000` | 仅显示GDP ≥ 5万亿的省 |

#### 核心 MapLibre 表达式

| 表达式 | 用途 | 本实验应用 |
|--------|------|----------|
| `match` | 枚举映射 | 7分类→7颜色 |
| `interpolate` | 线性插值 | 重要度→半径、高度→颜色 |
| `step` | 阶梯分段 | 聚合数量→圆大小/颜色 |
| `get` | 属性取值 | 获取 name/height/category |
| `has` | 属性存在 | filter 区分聚合/非聚合 |

#### 3D 建筑核心代码

```javascript
map.addLayer({
    id: 'buildings-3d',
    type: 'fill-extrusion',
    source: 'cbd-buildings',
    paint: {
        'fill-extrusion-height': ['get', 'height'],
        'fill-extrusion-color': [
            'interpolate', ['linear'], ['get', 'height'],
            0, '#ffeda0', 200, '#fc4e2a', 528, '#800026'
        ],
        'fill-extrusion-opacity': 0.85
    }
});
map.easeTo({ pitch: 60, bearing: -30, duration: 2000 });
```
