# 🗺️ 我的GIS作品集 — 功能说明

> **作者**：钟健武  
> **课程**：GIS工程与开发 · P1个人作品集  
> **技术栈**：Leaflet 1.9.4 + GeoJSON + MapLibre GL JS 4.7  

---

## 一、作品集概述

本作品集将课程第08-12讲所学的 **Leaflet 二维地图开发**、**GeoJSON 专题地图** 和 **MapLibre GL JS 矢量瓦片渲染** 三项核心技术整合为一个统一的个人 GIS 平台，用空间数据讲述个人成长故事。

### 设计理念

- **空间即是记忆**：每个地点都承载着一段人生经历，通过地图将零散的记忆组织为可探索的空间叙事
- **数据驱动表达**：用 GeoJSON 标准格式存储个人数据，以数据驱动的方式生成地图可视化
- **技术展示与实用结合**：每一项交互功能都既是课程技术的实践，也是真实可用的地图工具

## 二、数据内容设计

### 2.1 数据主题

本作品集以 **"我的人生轨迹"** 为主题，涵盖 17 个地点，分为 5 大类：

| 分类 | 图标 | 颜色 | 数量 | 说明 |
|------|:---:|:---:|:---:|------|
| 🎓 教育经历 | 🎓 | 蓝色 `#3b82f6` | 4 | 小学、中学、大学、GIS实验室 |
| 🏠 日常生活 | 🏠 | 红色 `#ef4444` | 3 | 老家、宿舍、图书馆 |
| ✈️ 旅行足迹 | ✈️ | 绿色 `#10b981` | 5 | 广州塔、故宫、外滩、兵马俑、成都 |
| 🍜 美食打卡 | 🍜 | 橙色 `#f59e0b` | 3 | 烧烤摊、早茶店、户部巷 |
| ❤️ 特别记忆 | ❤️ | 紫色 `#8b5cf6` | 2 | GIS竞赛、毕业答辩 |

### 2.2 数据覆盖

- **空间跨度**：覆盖 6 个省份/直辖市（广东、湖北、北京、上海、陕西、四川）
- **时间跨度**：2002 年（出生）~ 2025 年（毕业），横跨 23 年
- **数据格式**：GeoJSON FeatureCollection (RFC 7946)，属性包含 `name`、`category`、`date`、`description`、`icon`、`importance`、`memory` 等字段

### 2.3 数据来源

- 个人记忆与经历（坐标通过高德地图查询）
- 中国省级行政区划：阿里 DataV GeoAtlas API
- GDP 数据：2023 年国家统计局公报

## 三、功能详解

### 3.1 多图层管理（承接第08-09讲）

| 图层 | 说明 |
|------|------|
| 🗺️ OSM 标准底图 | 默认底图，OpenStreetMap 全球数据集 |
| 🎨 CartoDB 浅色 | 简约浅灰底图，适合数据可视化 |
| 🛰️ ESRI 卫星 | 高分辨率卫星影像 |
| 🌙 CartoDB 暗黑 | 暗色主题，护眼模式 |
| 🇨🇳 天地图 | 国家级地图服务（需 Key） |
| 📍 POI 标记层 | 17 个自定义分类图标标记 |
| 📊 GDP 专题层 | 34 省级行政区 YlOrRd 分级设色 |

通过右上角图层控制器一键切换底图，所有自定义图层（POI、专题）独立于底图存在。

### 3.2 三种探索模式

| 模式 | 说明 | 核心交互 |
|------|------|---------|
| 🔍 **探索模式** | 自由浏览所有地点，按分类筛选 | 点击标记 → Popup / 点击列表 → 飞行定位 |
| ⏱️ **时光模式** | 按时间顺序展示地点，连线构成人生轨迹 | 蓝色虚线 + 顺序数字标注 |
| 📊 **专题模式** | 叠加中国 GDP 分级设色图（YlOrRd 顺序色带） | 悬浮高亮 → 点击聚焦省份 + 数据弹窗 |

### 3.3 POI 交互系统（承接第09讲）

| 交互方式 | 效果 |
|---------|------|
| **点击标记** | 弹出信息卡片（名称、日期、描述、回忆、分类标签） |
| **悬停标记** | 显示地点名称 Tooltip |
| **分类筛选** | 侧边栏 6 个分类按钮，点击过滤标记 + 地点列表 |
| **地点列表** | 按重要性排序，点击项 → 地图飞行 + 打开 Popup + 高亮动画 |
| **图标区分** | 5 种分类各用不同颜色的圆形 DivIcon + emoji |
| **高亮状态** | 选中标记放大 + 脉冲动画，视觉反馈明确 |

### 3.4 信息面板（侧边栏）

| 面板 | 功能 |
|------|------|
| **作者卡片** | 姓名、专业、个人格言 |
| **分类筛选** | 6 个分类按钮（全部/教育/生活/旅行/美食/记忆），支持切换 |
| **足迹统计** | 地点总数、涉及城市数、跨越年份、分类数量——实时数据驱动 |
| **地点列表** | 按重要性排列，点击即飞，自动高亮当前项 |
| **地图书签** | 保存/恢复地图视角，localStorage 持久化存储 |

### 3.5 距离测量工具（承接第09讲练习2）

- **激活方式**：点击 `📏 测距` 按钮 或 按住 Shift 键点击地图
- **功能**：多点折线测距，每段显示距离标签，底部显示累计总距离
- **算法**：Haversine 球面距离公式（`R=6371000m`）
- **清除**：一键清除全部测量点和线

### 3.6 GDP 专题地图（承接第10讲）

- **数据源**：阿里 DataV GeoAtlas 全国省级行政区 GeoJSON（34 要素）
- **配色方案**：ColorBrewer YlOrRd 8 级顺序色带
  - 选择依据：GDP 是连续数值型数据，有低→高方向性，顺序色带通过亮度递减编码数值大小
- **交互**：悬浮高亮（边框加粗 + 置顶）、点击缩放到省份 + 显示 GDP 数据
- **图例**：8 级色块 + 数值范围 + 色带类型说明
- **分级策略**：手动分级（Manual Breaks），低值区密集以区分欠发达省份

### 3.7 MapLibre 3D 扩展（承接第11讲）

通过头部导航按钮可跳转到独立页面：
- 🏗️ **3D 视图** → MapLibre GL JS 页面（数据驱动样式、点聚合、3D 建筑拉伸）
- 📊 **专题图** → GeoJSON 专题地图页面（分级设色 + 比例符号 + 过滤）

### 3.8 时间线模式

- 按 `date` 字段升序排列所有地点
- 绘制蓝色虚线连接，标注顺序编号
- 形成"从出生到毕业"的人生空间轨迹

## 五、统一界面风格

### 设计系统

- **主色**：蓝紫渐变 `#4f6ef7` → `#7c3aed`（专业、现代、科技感）
- **分类色**：5 色定性色板（蓝/红/绿/橙/紫），与分类一一对应
- **卡片式布局**：圆角 `12px`、阴影层次（sm/md/lg）、白色卡片 + 淡灰背景
- **字体**：系统字体栈（PingFang SC / Microsoft YaHei），保证跨平台一致性
- **动画**：`cubic-bezier(0.4, 0, 0.2, 1)` 缓动，200ms 过渡
- **响应式**：≥900px 桌面（侧边栏 320px）、<900px 平板（上下布局）、<480px 手机（紧凑间距）

### 交互一致性原则

1. **即时反馈**：所有可点击元素有 hover 状态变化（颜色/阴影/位移）
2. **视觉焦点**：选中项有明显高亮（放大 + 脉冲动画），失焦自动还原
3. **渐进增强**：核心功能（浏览 POI）在无 JS 时降级为静态地图
4. **容错处理**：网络加载失败有提示 + 重试机制
